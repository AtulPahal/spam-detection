# Spam Email/SMS Detection

A simple machine learning web application built with Streamlit to classify whether an email or SMS message is Spam or Not Spam.

## Running the Application

This project uses `uv` for package management. To run the app locally, execute the following command:

```bash
uv run streamlit run main.py
```

## Features

- Pre-trained Machine Learning Model (`scikit-learn`)
- Text preprocessing (`nltk`)
- Simple web UI (`streamlit`)
